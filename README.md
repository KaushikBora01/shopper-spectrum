\# Shopper Spectrum



Customer Segmentation and Product Recommendation System



\## Project Overview

This project analyzes e-commerce transaction data to segment customers using RFM (Recency, Frequency, Monetary) analysis and recommends products using collaborative filtering.



\## Tech Stack

\- Python

\- Pandas

\- NumPy

\- Scikit-learn

\- Matplotlib

\- Seaborn

\- Streamlit



\## Features

\- Data cleaning and preprocessing

\- Exploratory Data Analysis (EDA)

\- RFM-based customer segmentation

\- KMeans clustering

\- Product recommendation system
